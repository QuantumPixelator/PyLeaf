# PyLeaf
A Python GUI app that uses PySide6 to display information about cannabis strains. Written in Python 3.11.5, and PySide6 5.xx

##
## Cross-Platform: 
` Windows`

` Linux`

` macOS`

##
## Dataset: 

The dataset used in this app is from Kaggle:
https://www.kaggle.com/datasets/kingburrito666/cannabis-strains/
##

#### Dependencies:
`pip install pyside6`
